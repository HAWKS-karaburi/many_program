## 📌 이 저장소는 무엇인가요?

이 저장소는 코드 보관함입니다.
완성된 제품을 보여주기보다, 개발 중에 나온 아이디어와 실험 결과를 정리해두는 데 초점을 둡니다.

필요할 때 바로 참고할 수 있도록, 가볍고 명확하게 유지하는 것을 목표로 합니다.

---

## 🔥 프로젝트 한 줄 설명

**여러 프로그램과 검증된 코드 조각을 조용히 축적해두는, 미니멀한 개발자 코드 금고입니다.**

## 🚀 사용 방법

저장소를 가져오고, 원하는 폴더를 열어서 바로 확인하면 됩니다.

```bash
git clone https://github.com/HAWKS-karaburi/many_program.git
cd many_program
```

이후 필요한 언어별 파일을 실행하거나, 특정 예제만 골라 참고하면 됩니다.
